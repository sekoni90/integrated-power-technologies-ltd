## Image Guidelines

- Format: JPG or PNG
- Recommended size: 800x600px for product images
- Hero image: 1920x1080px
- Optimize images for web (compress to reduce file size)
- Use descriptive filenames

## Current Setup

The website currently uses placeholder images. Replace them with actual Socomec UPS product images by:

1. Download official Socomec product images
2. Place them in the appropriate directories
3. Update image paths in components if needed

## Where Images Are Used
